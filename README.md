# Sing-a-Song
Sing a Song is a simple online tool, designed as an extension of the song guessing game.

# Description
Firstly, this web app allows you to choose the language and the quantity of given word. Then the words are chosen, one by one, while the players have to guess the songs containing them. The first player to sign a song, using a given word, is awarded with a point. The player with the highest number of point by the end of the game wins.

# Tools
Designed as a website for recreational pourposes using VS Studio, Github, Glitch.me.
